# **App Name**: Catalyst Catalog

## Core Features:

- Product Listing Display: Show products in a responsive grid or list view, displaying the product image, name, and price.
- Product Details Page: Navigate to a dedicated page on product click, presenting a larger image, name, description, and price.
- Category Management & Filtering: Organize products into categories and allow users to filter the product list by selecting a category.
- JSON API Endpoints: Implement backend APIs to fetch all products, products by category, and individual product details from a JSON database.
- Product Search: Enable users to search for products by their name.
- Basic Cart Functionality: Allow users to add products to a shopping cart, display the item count in the cart, and view selected products on a dedicated cart page.
- Frontend-Backend Integration: Seamlessly connect the frontend user interface with the backend APIs for data retrieval and manipulation.

## Style Guidelines:

- Primary color: A cool, professional blue (#2B8EEA) to convey reliability and a modern feel, chosen for its inviting yet sophisticated presence on a light background.
- Background color: A very light, desaturated blue (#F5F8F9) derived from the primary hue, providing a clean and subtle canvas that enhances readability.
- Accent color: A vibrant aqua-green (#0AA893), offering a refreshing contrast to the primary blue and designed to draw attention to interactive elements like buttons and links.
- Headlines and body text: 'Inter' (sans-serif) for its modern, clean, and highly readable design across various screen sizes, ensuring clarity for product information.
- Utilize simple, easily recognizable outline icons for navigation, search, cart, and category filtering to maintain a minimalist aesthetic.
- Employ a responsive grid system for product listings and a clear, single-column layout for product details, ensuring optimal viewing on both desktop and mobile devices. Maintain ample whitespace for a clean presentation.
- Incorporate subtle hover effects on product cards and buttons, along with smooth transitions for page changes and filtering, to provide a polished user experience.
